# Portfolio

A portfolio web application to operate as a 'sandbox' mode to help broaden and strengthen my skill set.
Deployed on Digital Ocean utilizing Nginx and PM2.

## Built With
```
Javascript
Nodejs
Express
Pug
PostgresSQL
Knexjs
Jquery
Bootstrap
SCSS
Lightbox
Google Analytics
PM2
Nginx
```
